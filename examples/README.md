## Receipt Format

Each receipt is a tamper-evident JSON record:

```json
{
  "exchange_id": "8f5df0cc245deee0",
  "timestamp": 1782124849.413,
  "model": "deepseek-chat",
  "user_message": "...",
  "assistant_response": "...",
  "claims": [{"label": "FACT", "text": "..."}, ...],
  "drift_score": 0.0741,
  "hash": "sha256:..."
}
```

The `hash` field seals the exchange — any modification to the response, claims, or metadata changes the hash. Receipts are append-only and auditable.
